# STM32F1系列单片机电赛备赛程序代码
## 代码清单

|序号|文件夹名称|添加的功能|
|----|---|----|
|1|1key_L|按键模块|
|2|2USART_L|串口通信模块|
|3|3Timer_L|定时器模块|
|4|4print_L|printf重定向（向串口终端打印）|
|5|5ADC_L|两路AD转换模数|
|6|6PWM_L|两路PWM输出模块|
|7|7gy61_L|三轴加速度模块|
|8|8PID_L|PID控制模块|

### 所有工程都是在 `上一个工程的基础上添加了 ` 新的外设功能。可以 根据自己项目的需要 选择对应的工程，修改代码中的 `管脚名 ` 即可。代码的运行环境 `keil MDK`。开发板 `STM32F1最小系统`
